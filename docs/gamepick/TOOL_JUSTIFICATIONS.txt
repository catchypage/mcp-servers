================================================================================
find_game
================================================================================

Read Only: Yes
Catalog-backed search and detail only. No writes to our DB; no user state mutated by the tool.

Open World: Yes
User-defined search query and optional game_id; external game metadata from the games catalog API (via our API routes).

Destructive: No
Read-only lookups.

Idempotent: No
Search results may vary as the catalog updates.

================================================================================
browse_top_games
================================================================================

Read Only: Yes
Returns a page of highly rated games (computed score, descending), optional genre filter.

Open World: Yes
User may pass genre slug and pagination offset for more pages.

Destructive: No

Idempotent: Yes
Same offset + genre returns the same ordered slice for a stable catalog state.
