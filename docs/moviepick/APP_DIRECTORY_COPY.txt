Description
================================================================================
MoviePick helps you discover movies and TV shows inside ChatGPT. Search by title with optional genres and year range, browse results with posters, and open full details (plot, cast, rating, runtime). Prefer a surprise? Use random mode with the same filters — movies, TV, or both. No sign-in required. Data comes from TMDB via our read-only API.
