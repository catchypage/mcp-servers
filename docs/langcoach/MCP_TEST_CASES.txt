LangCoach — MCP test cases (OpenAI Apps submission)
Copy one block per form field. Connector: POST https://justmatch.us/api/mcp/langcoach (after OAuth). Public tool for the model: open_lang_coach only. Placement test, profile, and settings run inside the widget after the app opens (not listed here).

Test Case 1 — Open the app to practice English
--------------------------------------------------------------------------------

Scenario
User explicitly asks to open Lang Coach to practice English in the embedded workspace.

User prompt
Open Lang Coach and help me practice English.

Tool triggered
open_lang_coach

Expected output
tools/call succeeds with assistant-visible text (message) and the Lang Coach widget loads (outputTemplate / embedded UI). User completes OAuth if needed, then uses the app inside the widget.
